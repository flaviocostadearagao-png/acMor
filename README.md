# PedalMatch 🚲 (Versão Estática Pura)

App de rastreamento e rede social para ciclistas. Esta versão foi simplificada para rodar **diretamente com HTML, CSS e JavaScript puros**, garantindo 100% de compatibilidade com o **GitHub Pages** sem necessidade de compilação.

## 🏗️ Estrutura do Projeto

-   **`index.html`**: A estrutura completa do aplicativo (Single Page Application).
-   **`main.js`**: Toda a lógica de integração com Firebase, Auth e Tracking.
-   **`firebase-applet-config.json`**: Configurações de conexão com seu banco de dados.

## 🛠️ Tecnologias
-   **Tailwind CSS**: Estilização via CDN.
-   **Firebase SDK**: Autenticação e Firestore via CDN.
-   **Lucide Icons**: Ícones minimalistas.

## 🚀 Como Rodar Localmente
1. Simplesmente abra o arquivo `index.html` no seu navegador.
2. Ou use um servidor simples: `npx serve .`

## 🌐 Deploy no GitHub Pages
Basta subir os arquivos diretamente para o seu repositório. O GitHub Pages servirá o `index.html` automaticamente.
